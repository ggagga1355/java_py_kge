초보 웹 개발자·HTML/CSS 학습 중

사용자가 보기 편한 화면을 만드는 것에 관심이 많다. 시맨틱 태그를 사용해서 구조를 명확하게 작성하려고 노력하고 있으며 간단한 레이아웃, 텍스트 스타일링,배경, 그림자 효과 등을 구현할 수 있다.

기술 스택/ 사용 가능 기술

HTML

  • 시맨틱 태그(header, main, section, footer)를 활용해 의미 있는 구조를 작성할 수 있다.
  • 이미지('img')와 링크('a'), 단락('p'), 인라인 텍스트('span')를 적절히 사용할 수 있다.

CSS

HTML

  • 그림자, 배치 할 수 있음
  • 테두리, 배경 할수 있음

프로젝트

포트폴리오 페이지 만들기
2025.10 진행중

시맨틱 태그와 CSS 기본 스타일만으로 개인 포트폴리오 페이지 제작했다. 박스 모델, 여백,테두리, 그림자 등을 이용해 카드 형태의 UI을 구성했고flex를ㄹ 이용해 프로필 영역의 ㅅ사진과 텍스트를 가로로 정렬했다.

간단한 소개 랜딩 페이지

한 화면에 "누구인가 / 무엇을 할 수 있는가/ 연락 방법"을 정리한 단일 페이지 형태로 구성했다. 헤더 영역에 큰 제목과 부제목을 두고, 본문은 카드 형태의 섹션으로 나누어 시각적으로 구분했다.
